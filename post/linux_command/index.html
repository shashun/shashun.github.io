Linux的常用命令--账号管理和文件管理

2019年4月27日
| Linux

1.账号管理

新增用户： useradd

$ useradd demo
$ useradd -g demo demo #指定用户组

修改密码：passwd

$ passwd demo #修改指定用户的密码 root用户有权限
$ passwd  #修改当前登录用户密码

删除用户：userdel

$ userdel demo

增加用户组：groupadd

$ groupadd demo

删除用户组：groupdel

$ groupdel demo  #如果已经有用户属于此组会删除失败

切换用户：su

$ su   #切换到root
$ su demo  #切换到demo

以其他用户的身份执行命令：sudo

$ sudo vi /etc/hosts

2. 文件管理

切换路径：cd

$ cd ~ #切换到家目录

创建目录：mkdir

$ mkdir demo

当前目录：pwd

$ pwd

删除目录：rmdir和rm

$ rmdir demo #必须是空目录
$ rm -rf demo

创建文件：touch

$ touch demo.txt

删除文件：rm

$ rm  demo.txt 
$ rm  -f demo.txt  #忽略确认
$ rm  -rf ./demo #删除目录下所有文件

移动文件：mv

mv demo.txt /tmp
mv  ./demo /tmp

复制文件或目录：cp

cp demo.txt demo1.txt #文件
cp -r a b #目录

查看文件：cat、head、tail

cat demo.txt 
head demo.txt 
head  -n 20 demo.txt  #指定显示行数
tail -n 20 demo.txt  
tail -f demo.txt

查看文件或目录详情： ls

ls -al

改变文件权限：chmod

u、g、o分别代表拥有者、拥有组、其他人
r、w、x分别代表读、写、可执行
r=4,w=2,x=1 

chmod u+x demo
chmod 777 demo
chmod -R 777 ./demo #递归更改

改变文件拥有者：chown

chown demo demo.txt
chown demo:demo demo.txt

改变文件拥有组：chgrp

chgrp demo demo.txt

一般查找：find

find PATH -name FILENAME
find / -name httpd.conf
find / -name *.conf
find / -name httpd*

数据库查找：locate

updatedb 
locate httpd.conf

查找执行文件：which/whereis

which passwd
whereis passwd # 不仅执行文件，还能找到相关的man文件

压缩与解压：gzip/gunzip、bzip2

gzip demo.txt
gunzip demo.txt.gz

bzip2 demo.txt
bzip2 -d demo.txt.bz2

文件备份：cpio

需要与find命令一起使用

#备份
find /etc -name *.conf| cpio -cov ./conf.cpio
#还原
cpio --absolute-filenames -icvu < ./conf.cpio

打包：tar

tar -zcvf  demo.tar.gz ./demo
# -c 创建压缩文件
# -z 使用gzip压缩
# -f 使用指定文件名
# -v 显示当前的文件

tar -zxvf  demo.tar.gz 
# -z 解压文件
# -x 从存档展开文件

软链接和硬链接：ln

ln demo.txt demo1.txt #硬链接
ln -s demo.txt demo1.txt #软链接

Linux的常用命令--字符处理、例行任务和网络管理与测试

2019年4月27日
| Linux

字符处理

搜索文本:grep

 grep [-ivnc] '需要匹配的字符' 文件名

 #-i不区分大小写
 #-c统计包含匹配的行数
 #-n输出行号
 #-v反向匹配
 grep -i 'name' ./demo.txt

 #使用cat命令+管道符
 cat ./demo.txt |grep -i 'name'

排序：sort

 sort [-ntkr] 文件名
 #-n采取数字排序
 #-t指定分隔符
 #-k指定第几列
 #-r反向排序

cat -t ':' -k 2 -n sort.txt
cat sort.txt | sort -t ':' -k 2 -n

删除重复内容

uniq [-ic]
#-i忽略大小写
#-c计算重复行数

cat uniq.txt | uniq

#使用-c参数就会在每行前面打印出该行重复的次数
cat uniq.txt | uniq -c

例行任务

单一时刻执行一次任务：at/atq/atrm

#30分钟后自动关机
$ at now + 30 minutes
at> /sbin/shutdown -h now
at> <EOT>
job 1 at 2012-11-06 23:39

#在午夜12点实现自动关机
$ at 00:00 2012-11-07
at> /sbin/shutdown -h now
at> <EOT>
job 2 at 2012-11-07 00:00

#查询at的任务
$ atq
1       2012-11-06 23:39 a root
2       2012-11-07 00:00 a root

#删除标号为1的任务
$ atrm 1

周期性执行任务：cron

cron服务必须时启动状态

service crond status
service crond start

crontab -e 进入编辑模式

*  *  *  *  *      command
#分钟 1～59
#小时 0～24 
#日期 1～31
#月份 1～12
#星期几 0～6

#星号（*）：代表所有可能的值，例如month字段如果是星号，则表示在满足其它字段的制约条件后每月都执行该命令操作。
#逗号（,）：可以用逗号隔开的值指定一个列表范围，例如，“1,2,5,7,8,9”
#中杠（-）：可以用整数之间的中杠表示一个整数范围，例如“2-6”表示“2,3,4,5,6”
#正斜线（/）：可以用正斜线指定时间的间隔频率，例如“0-23/2”表示每两小时执行一次。同时正斜线可以和星号一起使用，例如*/10，如果用在minute字段，表示每十分钟执行一次。

其他参数

crontab -l # 查看所有任务
crontab -r # 删除所有任务
crontab -u demo -l #root用户可以查看其他用户的任务

/etc/crontab的管理

*  *  *  *  *   user   command
#第六列定义了以什么身份执行例行任务

网络管理

检查和配置网卡：ifconfig/ip addr


#配置IP
ifconfig eth0 192.168.159.130 netmask 255.255.255.0
ifconfig eth0 192.168.159.130/24 #通过IP地址和掩码系统能自行算出广播地址，也可以显式地指定广播地址
ifconfig eth0 192.168.159.130 broadcast 192.168.159.255  netmask 255.255.255.0

#关闭
ifconfig eth0 down
ifdown eth0
#启用
ifconfig eth0 up
ifup eth0

将IP写入配置文件

RedHat和CentOS系统的网络配置文件所处的目录为/etc/sysconfig/network-scripts/

文件名 ifcfg-eth0

    [PCnet32 LANCE]
    DEVICE=eth0
    BOOTPROTO=dhcp
    ONBOOT=yes

    [PCnet32 LANCE]
    DEVICE=eth0
    BOOTPROTO=static
    ONBOOT=yes
    IPADDR=192.168.159.129
    NETMASK=255.255.255.0

DNS配置

 #/etc/resolv.conf
 nameserver 192.168.159.2
 #nameserver关键字后面紧跟着一个DNS主机的IP地址，可以设置2~3个nameserver

网络测试工具

ping命令

ping [-ciw] 域名或IP
#c  ping的次数
#i  ping包的发送间隔
#w  ping没有回应，在指定的时间退出
